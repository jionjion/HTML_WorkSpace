# 介绍
为 JavaScript 的超集合, 使用时需要通过 Webpack 翻译为 JavaScript 脚本

## 优点
1. 规范代码,强类型
2. 编译阶段发现问题
3. 支持ES最新特性

## 语法
1. 支持ES6等扩展语法
2. 支持强类型转换
3. 需要编译器,生成原生JavaScript文件

## 配置环境
官网 https://www.typescriptlang.org/

安装 npm install -g typescript

## 工具链
### tsc 命令
查看当前版本 
tsc -v

编译文件    
tsc .\main.ts

运行编译后生成的JavaScript脚本
node main.js


## 工作流
1. 编写 *.ts 文件
2. 交由TypeScript编译器编译成为JavaScript文件
3. 浏览器使用 *.js 文件
