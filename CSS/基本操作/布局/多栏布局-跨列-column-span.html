栏布局column-span

column-span主要用来定义一个分列元素中的子元素能跨列多少。
column-width、column-count等属性能让一元素分成多列，不管里面元素如何排放顺序，他们都是从左向右的放置内容， 但有时我们需要基中一段内容或一个标题不进行分列，也就是横跨所有列，此时column-span就可以轻松实现，此属性的语法如下。

column-span: none | all
none:此值为column-span的默认值，表示不跨越任何列。
all:这个值跟none值刚好相反，表示的是元素跨越所有列，并定位在列的Ｚ轴之上。
