# 项目介绍
## 安装
### Vue环境
```
npm install -g @vue/cli
```

### 初始化服务
```
npm install -g @vue/cli-init
```

### 控制服务
```
npm install -g @vue/cli-service-global
```


## 启动项目
### 使用UI管理项目
```
vue ui
``` 

### 命令行初始化项目
项目名不能有大写,根据交互提示输入内容或者直接回车默认初始化项目
```
vue init webpack vue-example
```

### 安装项目依赖
到项目路径下,执行
```
cnpm install
```

### 运行项目
在UI管理界面中启动
